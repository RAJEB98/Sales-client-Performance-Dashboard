🔹 Project Overview

This project focuses on analyzing sales data using SQL and visualizing key business insights through an interactive Power BI dashboard. The goal is to transform raw data into meaningful insights that help understand business performance, revenue trends, and profitability.

The project combines data analysis (SQL) and data visualization (Power BI) to deliver a complete end-to-end analytics solution.

🔹 Tools & Technologies Used
*SQL (MySQL)
*Power BI
*DAX (for calculations)
*Data Cleaning & Transformation

🔹 Learning Outcomes
*Gained hands-on experience in SQL for real-world analysis.
*Learned advanced SQL concepts like window functions.
*Improved ability to derive business insights.
*Built interactive and professional dashboards.
